top
トップページを表示します。
about
blogと執筆者についての情報を表示します。
recent
最近更新のあった記事から順に表示します。
list
blog内の全記事を一覧表示します。
search
blog内の記事を検索するためのフォームを開きます。
newsfeed
ニュースフィードのURLと、フィードについての簡単な説明を表示します。
mastermenu
管理者メニューを開きます。（ログイン中のみ有効）
login
管理者としてログインするためのフォーム画面を表示します。（非ログイン中のみ有効）
logout
管理者モードを解除します。（ログイン中のみ有効）
addentry
write
管理者メニューの「新しい記事を書く」と同じです。（ログイン中のみ有効）
entry [記事のID]
指定した記事（エントリ）を表示します。
entries [記事1のID] [記事2のID] ...
指定した記事（エントリ）を表示します。entryコマンドと違い、複数の記事を並べて表示することが可能です。
uri [URL]
url [URL]
指定したURLへリンクします。
  • メニュー定義についての解説（Pinky:blog配布サイト内）
